## Project Overview
This project analyzes parliamentary constituency-wise voter turnout data across 2014, 2019, and 2024 elections. The dataset was cleaned in Excel, converted to CSV format, and imported into Power BI for interactive dashboard development.

## Tools Used
- Microsoft Excel (Data cleaning & transformation)
- Power BI
- DAX Measures
- Data Modeling

## Dataset
- voter_turnout_raw_data.csv

## Dashboard Features
- Year-wise voter turnout trend
- Gender-wise turnout comparison
- Constituency-level analysis
- Interactive slicer filtering
- Heatmap-style comparison table

## Key Insights
- Overall turnout fluctuated across election years.
- Female voter participation increased steadily.
- Some constituencies showed consistently low turnout.
- Postal votes contributed a small percentage of total turnout.
